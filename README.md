# Запуск проекта

Скачать проект, распаковать архив, в корне проекта ввести команду:

```bash
npm i
````

## Dev-версия

Для запуска дев-сборки ввести:

```bash
npm run dev
```

## Prod-версия

Для запуска прод-сборки ввести:

```bash
npm run build

npm run start
```

Проект разворачивается на 3000 порту.
